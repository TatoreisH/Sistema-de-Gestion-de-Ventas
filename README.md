## Sistema de Gestion de Ventas

Arquitectura de software para un Sistema de Gestión de Ventas que aplica el patron arquitectónico Model-Vista-Controladora.

## Funcionalidades
- Gestion de ventas
- Gestion de clientes
- Gestion de productos
- Reportes
- Consultas.

## Tecnologias
- C#
- SQL
- Entity Framework
- Patron MVC
- Visual Studio 2022
